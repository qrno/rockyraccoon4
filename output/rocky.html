Rocky Raccoon

what is Rocky Raccoon?

Rocky Raccoon is a static website generator written in Python. It takes in files in a fictional markup language called rocky and, given a template, generates a html file. This blog is generated using Rocky Raccoon.

On the rocky files you can set data that can be usd on the template (for example on the html title tag.)

features

Rocky Raccoon currently has support for the followigng tags: section (which is converted to h1), subsection (h2), link (a), image (img), paragraph (p), bold (b) and italic (i).
